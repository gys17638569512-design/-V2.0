# 数字化维保系统 · Digital MMS v1.2

起重机维保 SaaS 平台 — PC 管理后台 UI 原型

## 功能模块

- **工作台** — 全局数据看板，逾期工单、设备预警、工程师动态、完工趋势
- **工单中心** — 工单列表、新建派单（三步流程）、工单详情
- **客户管理** — 客户列表、客户预览抽屉、合同信息
- **设备档案** — 三编号管理（系统编号/出厂编号/场内编号）、设备详情五Tab
- **报告中心** — 报告列表、报告预览、签字状态管理
- **系统设置** — 公司信息、检查模板矩阵

## 技术栈（目标）

- 前端框架：Vue 3 + Element Plus
- 状态管理：Pinia
- 网络请求：Axios
- 后端 API：FastAPI（Python 3.10+）

## 快速预览

直接用浏览器打开 `index.html` 即可查看完整 UI 原型，无需安装任何依赖。
